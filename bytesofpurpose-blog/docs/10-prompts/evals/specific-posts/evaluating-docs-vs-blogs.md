# Docs vs Blogs Rubric

**Blog Post**: `2025-01-31-docs-vs-blogs.md`  
**Focus**: Content strategy and format selection

## Core Questions

### ✅ Problem Clarity
- [ ] **Clear distinction**: Does the post clearly explain the difference between docs and blog posts?
- [ ] **Use case examples**: Does it provide specific examples of when to use each format?
- [ ] **Decision framework**: Does it offer a practical framework for choosing between them?

### ✅ Practical Value
- [ ] **Actionable guidance**: Can readers immediately apply the decision criteria?
- [ ] **Real examples**: Does it include concrete examples of good docs vs good blog posts?
- [ ] **Framework utility**: Is the decision framework easy to use and remember?

### ✅ Content Quality
- [ ] **Clear writing**: Is the content easy to understand and follow?
- [ ] **Logical structure**: Does it flow logically from problem to solution?
- [ ] **Engaging tone**: Is it written in an engaging, helpful style?

## Specific Criteria

### Content Strategy Focus
- [ ] **Format comparison**: Clearly compares docs vs blog posts
- [ ] **Decision criteria**: Provides clear criteria for choosing between formats
- [ ] **Examples**: Includes specific examples of each format
- [ ] **Framework**: Offers practical decision-making framework

### Value Delivery
- [ ] **Immediate applicability**: Readers can use the guidance right away
- [ ] **Clear benefits**: Shows why format choice matters
- [ ] **Practical tips**: Provides actionable advice for content creators

### Writing Quality
- [ ] **Scannable format**: Uses headers, bullets, and clear structure
- [ ] **Concise explanations**: Avoids unnecessary complexity
- [ ] **Engaging examples**: Uses relatable scenarios and examples

## Quality Threshold

**Ready to Publish**: 8+ criteria met (80%+)  
**Needs Revision**: 6-7 criteria met (60-79%)  
**Major Revision**: Below 6 criteria met ("under 60%")

## Key Questions to Ask

1. **Would a content creator find this immediately useful?**
2. **Does it solve a real problem content creators face?**
3. **Is the decision framework practical and memorable?**
4. **Are the examples clear and relatable?**
