**Blog Post**: `2025-09-25-understanding-zapier-values.md`  
**Focus**: Company culture analysis and insights

## Core Questions

### ✅ Cultural Analysis
- [ ] **Value breakdown**: Does it clearly explain each of Zapier's 5 core values?
- [ ] **Practical examples**: Shows how values translate to actual behaviors?
- [ ] **Cultural insights**: Provides meaningful analysis of the culture?

### ✅ Comparative Analysis
- [ ] **Company comparison**: Does it compare Zapier to other tech companies?
- [ ] **Size context**: Shows how company size affects cultural values?
- [ ] **Unique positioning**: Identifies what makes Zapier's approach distinctive?

### ✅ Practical Value
- [ ] **Applicable insights**: Can readers apply these insights to their own work?
- [ ] **Decision framework**: Helps readers understand different cultural models?
- [ ] **Actionable takeaways**: Provides concrete guidance for cultural decisions?

## Specific Criteria

### Cultural Understanding
- [ ] **Value explanation**: Clearly explains each of the 5 core values
- [ ] **Behavioral examples**: Shows how values translate to specific behaviors
- [ ] **Cultural context**: Provides background on why these values matter
- [ ] **Evolution story**: Shows how values have developed over time

### Comparative Analysis
- [ ] **Company comparison**: Compares Zapier to Amazon, Meta, or other companies
- [ ] **Size analysis**: Shows how company size influences cultural development
- [ ] **Unique insights**: Identifies what makes Zapier's approach distinctive
- [ ] **When to use**: Provides guidance on when Zapier's model applies

### Practical Application
- [ ] **Remote work insights**: Shows how values support remote-first culture
- [ ] **Team building**: Provides guidance for building similar cultures
- [ ] **Decision making**: Helps readers choose between cultural models
- [ ] **Implementation**: Offers practical steps for applying insights

## Quality Threshold

**Ready to Publish**: 8+ criteria met (80%+)  
**Needs Revision**: 6-7 criteria met (60-79%)  
**Major Revision**: Below 6 criteria met ("under 60%")

## Key Questions to Ask

1. **Does this provide deep insights into Zapier's culture?**
2. **Would someone considering working at Zapier find this valuable?**
3. **Does it help readers understand different cultural approaches?**
4. **Are the insights actionable and applicable beyond Zapier?**
