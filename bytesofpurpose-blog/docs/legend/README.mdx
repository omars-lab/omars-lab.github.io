# 🧭 Legend: How This Site Works

This is the front door, the legend for the whole map. It explains the one idea the site
is organized around (durable vs temporal), what the little emoji next to each post title
mean, the kinds of posts I write, and where to find the glossaries. The sidebar holds the
rest: the glossary and every terminology page.

## Durable vs temporal: the one idea

Everything here is split into two halves by a single question: **does this stay true over
time, or is it a dated thing I did?**

- **Craft and Journey are DURABLE.** Distilled learnings, frameworks, strategy, terminology,
  the way I see the world (Craft) and myself (Journey). Return-to knowledge. When something I
  learn holds up, it lands here.
- **Initiatives are TEMPORAL.** The specific, dated things I actually did: experiments,
  project logs, posts. Each one links up to the durable Craft insight it informed.

So if you want the *lasting idea*, read Craft or Journey. If you want *what I actually did
and when*, read Initiatives. A temporal post is the evidence; a durable page is the lesson.

## The emoji tell you the TYPE of post

Every post leads with an emoji in the Posts list. The emoji is not decoration and it is
not about the topic. It tells you what KIND of thing the post is, so you can scan the
list and find the format you are in the mood for. Here is the legend:

| Emoji | Kind | What it is |
|---|---|---|
| ❓ | Question set | A set of questions to ask yourself, grouped by theme. The "What I Ask Myself" series. |
| 🧩 | Framework | A reusable model for thinking about something. "A Framework for", "The Anatomy of". |
| 💭 | Reflection | A personal essay or set of takeaways. "Notes From", "What X Taught Me", affirmations. |
| 🏗️ | System design | An architecture or build write-up: how something was designed and shipped. |
| 📐 | Design story | The story behind a design: why it exists, with a link to the full design doc. |
| 🧪 | Tutorial | A hands-on, learn-by-doing walkthrough. "A Hands-On Way to", "Running X Locally". |
| 📖 | Reference | A concept explainer or taxonomy. "A Taxonomy of", "What Does X Mean". |
| 🎙️ | Event recap | Notes from a conference or event. |
| 🧭 | Legend | A guide like this one: it indexes a set of posts and the conventions they share. |
| 📚 | Learning plan | A sequenced plan for learning a topic: the goal, an ordered curriculum, resources, and how you know you're done. |
| 📝 | Experiment plan | The plan half of an experiment: the hypothesis and the design, before the result is in. |
| 📊 | Experiment result | The result half of an experiment: the outcome and the decision, once the data is in. |

The title in the sidebar is kept short on purpose. The full title still lives at the
top of each post and in the browser tab.

## Where to start

A few entry points depending on what you came for:

- **To sit with a question about yourself**, start with the question sets (❓). The
  keystone is [What I Ask Myself](/initiatives/what-i-ask-myself), which explains the
  practice and the power icons on each question.
- **To borrow a way of thinking**, browse the frameworks (🧩).
- **To see how something was built**, read the system designs (🏗️) and tutorials (🧪).
- **For a shorter, more personal read**, the reflections (💭) are the place.

## The glossary and terminology

When a post uses a term of art, it is defined here in the Legend. The sidebar holds:

- **[The Glossary](/legend/glossary)**: the single A-to-Z index of every term, linking each
  to its definition.
- **Terminology**: the per-topic term pages (software development, productivity, companies),
  plus the emoji and acronym conventions.

## How the question sets work

The largest group on the blog is the question sets. Each one is a themed collection of
questions I actually return to, with a power icon on every question showing what kind
of charge it carries (a jolt, a burn, something that reshapes you over time). The full
legend for those power icons lives in the keystone post,
[What I Ask Myself](/initiatives/what-i-ask-myself). If you only read one thing, read that.
