# 🎯 Interviewing

The mental model behind interviewing: knowing the target before you prep. Companies hire
against a **level** with rising expectations and evaluate a consistent set of
**competencies**, so [understanding what they expect](/knowledge/interviewing/understanding-what-companies-expect)
tells you what to demonstrate and at what bar.

For the step-by-step walkthrough of the interview itself, from application to offer, see
[The Process of Interviewing](/craft/interview-prep/process-interview) in Interview Prep.
