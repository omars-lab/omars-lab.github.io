# Software & Tooling Ideas

A running backlog of things I want to build, grouped by theme. This is the raw front of the
idea-to-ship lifecycle: capture now, groom later, and promote the ones worth doing into real
initiatives. Every open capture is preserved here (only items already done, and dead
local-only links, were pruned); when one is worth doing it gets matured into a board-ready idea.
