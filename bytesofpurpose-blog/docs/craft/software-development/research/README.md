# Research

This is the durable index of what I want to **learn**: the topics, tools, and frameworks I
am reading up on, and the questions I need answered before I act. Each entry is a **dated
learning log** on the [Initiatives](/initiatives) blog. This page groups them by area.

These same learning logs also flow onto the [Research board](/craft/product-management/research)
by stage (backlog, researching, learned). This hub is the view **by area**; the board is
the view **by progress**. Move an entry between areas by editing its `area` frontmatter, and
it re-sorts here automatically.
