# Tinkering

This is the durable index of what I have **tinkered** with: hands-on dabbling with a tool,
language, or technique, short of a finished build. Each entry is a **dated tinkering log**
on the [Initiatives](/initiatives) blog. This page groups them by area so you can see the
whole spread of experiments at once.

Same line as everywhere on the site: a tinkering *log* is temporal (a dated thing I poked
at), so it lives on Initiatives; this hub is durable (the return-to index), so it lives
here in Craft. Move an entry between areas by editing its `area` frontmatter, and it
re-sorts here automatically.
