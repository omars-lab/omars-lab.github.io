# Repos of Interest

A running catalog of repositories worth returning to: tools I reach for, projects that are
relevant to my work, and reference implementations I learn from. Each repo is tagged by
what it is, and the catalog groups them by tag, so a repo can show up in more than one place
(a testing tool is both `#tool` and `#testing`).

To add one, edit `src/data/repos-of-interest.json`; the catalog below regenerates from it.
