# Prompts

This is the durable index of the reusable **AI prompts** I have built: the prompt systems I return to
for knowledge work, content creation, productivity, and evaluation. Each is a **dated post** on the
[Initiatives](/initiatives) blog that explains what the prompt does, the value it delivers, and links
to the prompt itself. This page groups them by area so you can scan the whole toolkit at once.

Like every hub on the site, a prompt *log* is temporal (a dated thing I built), so it lives on
Initiatives; this index is durable, so it lives here in Craft. Move a prompt between areas by editing
its `area` frontmatter, and it re-sorts here automatically.

## What's here

- **Reusable prompts** that automate a workflow: analyzing SQL, authoring blog posts, maintaining the
  site, refactoring roles, carrying over todos, customizing kanban boards, organizing personal content.
- **Evaluation rubrics** (`eval-*`): focused question sets that gate a blog post from draft to
  published, both universal rubrics and per-post ones.
