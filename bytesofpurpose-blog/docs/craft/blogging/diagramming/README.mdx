# Diagramming

Techniques for creating **visual diagrams from text** so your documentation stays
version-controlled, diff-able, and easy to maintain. Text-based diagrams live next to
your code, review like code, and never drift out of sync the way exported images do.

## What You'll Find Here

- Text-based diagram creation methods
- Diagramming with PlantUML
- Icon sourcing and implementation
