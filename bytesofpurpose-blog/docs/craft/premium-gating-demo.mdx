## The gated body

If you are reading this sentence on the live site without signing in, the gate is
broken — this paragraph contains the sentinel **PREMIUMSENTINELBODY** that the build
must encrypt away. Signed-in readers see it decrypted in place.

- Premium bodies are encrypted at MDX-compile time, so the plaintext is in neither the
  HTML nor the JS bundle.
- The Cloudflare Worker vends the decryption passphrase only to a valid LinkedIn session.
- The encrypted payload ships as a sidecar JSON next to the page.

```js
const proof = 'PREMIUMSENTINELCODE never ships in cleartext';
```
