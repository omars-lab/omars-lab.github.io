# Arch Illustrations

Imagery is the hand-drawn **warm cartoon** set: cozy brown-and-green scenes (a coder at a desk with
coffee, prayer, sculpting a brain) clipped into a **cathedral-arch silhouette**. Warm, narrative,
personal, never stock photography and never gradient abstraction.

## The cathedral-arch silhouette

Every scene is clipped to the same arch: `border-radius: 110px 110px 14px 14px`, with an outlined
edge and a soft drop-shadow (`--shadow-arch`). The arch is the brand's signature frame; it echoes
the faith-meets-craft theme and matches the homepage chooser cards.

## The scene set

Each scene names a part of the site. They are the same illustrations used on the homepage chooser
and the section cards.

## Adding a new scene

A raw image has to be fitted to the **canonical arch** (1024², a fixed bounding box, transparent
outside, zero fringe) before it drops in cleanly. The repo's `import-arched-image` workflow detects,
scales, shifts, and clips a raw image to that arch and writes a new file, so a new scene matches the
set exactly. Never crop in place; a white-line artifact at the arch edge is a canonical-arch
violation, so re-fit rather than tweak CSS.
