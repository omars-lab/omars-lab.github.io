# Core Components

The design system's core set is **Button, Tag, Badge, Card, and Callout**. Buttons have their own
[Buttons page](/design-system/buttons) and the Card is demonstrated in the
[Components reference](/handbook/components/structural/card); this page covers the small chips and
the callout, rendered **live from the brand tokens**.

## Tags

A **tag** is a pastel pill: a small category chip. It is the one correct use of a pastel, a **fill**
with `--tea-ink` (deep green) on top. Never set a text color to a pastel.

## Badges

A **badge** is a small status marker: tighter than a tag, with its own micro-tracking. The default
badge is green; the gold badge marks premium.

## Callouts

A **callout** is the brand's admonition: a green-tinted wash with a primary-green left rule. The
gold callout is reserved for premium or insider tips.

<DemoCallout title="Note">
  A callout draws the eye to a short aside without shouting. It uses the accent tint and the green
  left rule, matching the site's admonition style.
</DemoCallout>

<DemoCallout title="Premium tip" gold>
  The gold callout is reserved for premium and insider tips, the same gold family as the premium
  gate and the lock badges. It appears nowhere else.
</DemoCallout>

:::note These are design references
These chips and callouts are rendered from the real design tokens to show the brand's intended
shapes. Some map to shipping components (the `<Card>` in the Components reference, the site's
admonitions); others, like the standalone Tag and Badge, are token-accurate references rather than a
single importable component. The [claude.ai design project](https://claude.ai/design) carries the
prototype React versions of the full core set.
:::
