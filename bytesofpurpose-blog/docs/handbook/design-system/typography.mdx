# Typography

Headings are **Fraunces**, an old-style display serif with an optical-size axis; it reads best
around weight 600 and wants only the faintest negative tracking. Body and UI are **Geist** at 16px
and a generous **1.7 line-height** for long-form comfort. Code is **Geist Mono**. Each specimen
below renders in the real face, with the tokens that produced it shown underneath.

## Display serif · Fraunces

The editorial voice. Large headlines take a hair of negative tracking; the subtitle is an italic
serif aside.

## Body sans · Geist

The workhorse for UI and long-form prose. Set at 16px with a 1.7 line-height for comfortable
reading.

<TypeSpecimen family="--font-sans-body" size="--text-body-md" weight="--weight-regular" tracking="--tracking-normal" leading="--leading-body" sizeLabel="16px / 1.7" sample="Real-world engineering artifacts. Examples include sequence diagrams, roadmaps, code snippets, and automations. Short, concrete, verb-first." />

## Eyebrow and mono

The eyebrow label is uppercase Geist, tracked out to `0.18em`, in the primary green. The middot
`·` is the brand's connective separator. Code is Geist Mono.

<TypeSpecimen family="--font-sans-body" size="--text-eyebrow" weight="--weight-semibold" tracking="--tracking-eyebrow" leading="--leading-tight" sample="ENGINEERING · FAITH · CRAFT" />

<TypeSpecimen family="--font-mono" size="--text-body-md" weight="--weight-regular" tracking="--tracking-normal" leading="--leading-body" sample="const purpose = 'one byte at a time';" />

:::note The eyebrow tracking is editorial-only
`--tracking-eyebrow` (`0.18em`) is for the hero and section eyebrow labels. Badges, pills, tags,
and chips keep their own tighter micro-tracking; do not normalize those to `0.18em`, it looks broken
on a small chip.
:::

## The type scale

Headings (Fraunces) and body (Geist) each walk a fixed scale. Reach for a scale step, not a raw
`rem`.
