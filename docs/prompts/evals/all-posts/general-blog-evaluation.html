General Blog Post Evaluation

Applicable to: All blog posts
Focus: Content quality, value proposition, and professional positioning

Content Quality Evaluation‚Äã

Writing Quality (20 points)‚Äã

  • Clear narrative structure (4 points)
  • Specific, concrete examples (4 points)
  • Actionable insights (4 points)
  • Technical accuracy (4 points)
  • Engaging writing style (4 points)

Value Proposition (15 points)‚Äã

  • Relevant to target audience (3 points)
  • Timely and current (3 points)
  • Unique perspective or approach (3 points)
  • Practical applicability (3 points)
  • Shareable content (3 points)

Professional Positioning (10 points)‚Äã

  • Demonstrates expertise (2 points)
  • Shows problem-solving ability (2 points)
  • Demonstrates leadership (2 points)
  • Shows collaboration skills (2 points)
  • Demonstrates learning mindset (2 points)

  • Needs Major Revision (Below 27 points)

Post-Evaluation Actions‚Äã

If Ready to Publish‚Äã

  • Final proofread and edit
  • Add appropriate tags and metadata
  • Schedule publication
  • Prepare social media promotion

If Needs Minor Revision‚Äã

  • Address specific feedback areas
  • Strengthen weak sections
  • Add missing examples or details
  • Re-evaluate after revisions

If Needs Major Revision‚Äã

  • Identify core issues to address
  • Consider restructuring content
  • Add substantial new content
  • Re-evaluate from scratch after major changes

Usage Instructions‚Äã

When to Use This Evaluation‚Äã

  • All blog posts should be evaluated using this general rubric
  • Combined with specific rubrics for comprehensive evaluation
  • Standalone evaluation for posts without specific rubrics
  • Quality gate before publishing any content

How to Use‚Äã

  1. Read the blog post carefully
  2. Check each criterion and assign points
  3. Calculate total score (45 points maximum)
  4. Apply quality thresholds to determine readiness
  5. Complete feedback sections with specific guidance
  6. Follow post-evaluation actions based on score

Integration with Specific Rubrics‚Äã

  • Use alongside specific blog post rubrics for comprehensive evaluation
  • Add scores together for complete assessment
  • Apply both thresholds for final decision
  • Combine feedback from both evaluations
